# Physics Prompt

Measure the partial width ratios R_b and R_c, and the b-quark
forward-backward asymmetry A_FB^b, in hadronic Z decays using
archived ALEPH data at sqrt(s) = 91.2 GeV.

Tag b and c events using lifetime methods (signed impact parameter
significance). Use a double-tag method for R_b. For A_FB^b, determine
the b-quark charge using jet charge in tagged hemispheres and measure
the cos(theta_thrust) asymmetry to extract sin^2(theta_eff).

If you lack MC truth info, you could derive approximate truth from
studying the decay chain. Maybe you can find neutrino truth flavour.

Look up most recent work on R_b, R_c, and A_FB^b to get the latest
conventions and methodologies. Query the LEP corpus for ALEPH R_b,
jet charge, and A_FB^b.

## Data

- Data: /n/holystore01/LABS/iaifi_lab/Lab/sambt/LEP/ALEPH/
- MC:   /n/holystore01/LABS/iaifi_lab/Lab/sambt/LEP/ALEPHMC/

## Observables

1. **R_b = Gamma(Z->bb) / Gamma(Z->hadrons)** — double-tag method with
   lifetime b-tagging (signed impact parameter significance)
2. **R_c = Gamma(Z->cc) / Gamma(Z->hadrons)** — lifetime tagging
3. **A_FB^b** — forward-backward asymmetry of b quarks using jet charge
   in tagged hemispheres, measured via cos(theta_thrust) asymmetry,
   extracting sin^2(theta_eff)

## Key Methods

- Signed impact parameter significance for b/c tagging
- Double-tag technique for R_b (hemisphere correlations)
- Jet charge for b-quark charge determination
- cos(theta_thrust) asymmetry for A_FB^b
